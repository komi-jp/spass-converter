# Contributing

Thanks for your interest in spass-converter! Contributions are welcome.

## Ways to contribute

- **Report bugs** — Open an issue with steps to reproduce
- **Request features** — Suggest new export formats or improvements
- **Submit pull requests** — Fork the repo, make your changes, and open a PR

## Security reminder

When reporting issues, please **never** include:
- Your `.spass` export files
- Passwords or decryption keys
- Exported CSV contents or any credential data

## Development setup

```bash
git clone https://github.com/komi-jp/spass-converter.git
cd spass-converter
pip install -r requirements.txt
```

## Pull request guidelines

- Keep changes focused and minimal
- Test with a real `.spass` file before submitting
- Update README.md if adding new features or formats
